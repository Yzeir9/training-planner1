## Описание

Приложение для планирования тренировок с графическим интерфейсом (Tkinter). Позволяет добавлять, удалять, просматривать и фильтровать тренировки. Все данные сохраняются в JSON-файл и загружаются при следующем запуске.

## Функциональность

- Добавление тренировки: дата, тип (выпадающий список), длительность в минутах.
- Просмотр всех тренировок в таблице.
- Фильтрация:
  - по типу тренировки (из выпадающего списка)
  - по дате (в формате ГГГГ-ММ-ДД)
- Удаление выбранной тренировки.
- Валидация ввода:
  - дата только в формате `YYYY-MM-DD`
  - длительность > 0 (целое или дробное число)
- Автоматическое сохранение в `training_data.json`.
- Кнопка сброса фильтров.

## Технологии

- Python 3.x
- Tkinter (встроенная библиотека)
- JSON для хранения
- Datetime для работы с датами
